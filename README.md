# P Sai Pratyusha's Portfolio

## Overview
This is a personal portfolio website for P Sai Pratyusha, showcasing skills in UI/UX Design and AI/ML. The website is built using HTML, CSS, and JavaScript, featuring a modern dark theme with responsive design.

## Features
- **Welcome Section**: Introduction to the portfolio
- **About**: Personal background and professional summary
- **Skills**: Comprehensive list of technical skills
- **Education**: Academic background
- **Internships**: Detailed internship experiences
- **Projects**: Showcase of key projects with images
- **Contact**: Contact information and contact form

## Technologies Used
- HTML5
- CSS3 (with gradients, animations, and responsive design)
- JavaScript (for smooth scrolling and interactivity)
- Google Fonts (Roboto)

## How to View
1. Open `index.html` in any modern web browser
2. The website is fully responsive and works on desktop, tablet, and mobile devices

## File Structure
- `index.html` - Main HTML file
- `styles.css` - Styling and layout
- `script.js` - JavaScript for interactivity
- `profile image.jpg` - Profile picture
- Project images (PNG/JPG files)
- `readme.txt` - This file

## Customization
To personalize this portfolio for yourself:
1. Replace content in `index.html` (name, bio, skills, projects, etc.)
2. Update contact information in the contact section
3. Replace `profile image.jpg` with your photo
4. Update project images and descriptions
5. Modify colors in `styles.css` if desired
